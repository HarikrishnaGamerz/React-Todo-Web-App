# React-Todo-Web-App
A clean, responsive, and functional Todo application built with React. This project demonstrates core frontend development concepts including state management, component lifecycle, and persistent data storage.

#Features
Create: Add new tasks to your list instantly.
Read: View all your tasks in a clean, organized list.
Update:Edit the text of existing tasks.
Toggle completion status (mark as done/undone).
Delete: Remove tasks from the list permanently.
Persistence: Uses localStorage to ensure your tasks remain saved even after refreshing the page or closing the browser.
Responsive UI: Optimized for both desktop and mobile viewing.

Tech Stack
Framework: React.js
Styling: CSS3 (Flexbox, Custom Properties)
Storage: Browser LocalStorage API

What I Learned
Building this project helped reinforce:
React Hooks: Using useState for UI logic and useEffect for side effects (syncing with storage).
Immutability: Handling state updates correctly using the spread operator and array methods like .map() and .filter().
Conditional Rendering: Switching between "View mode" and "Edit mode" within the same component.
Data Persistence: Converting JSON data to strings for storage and back to objects for application use.
